# Linkage-Mechanism-Laboratory
连杆机构实验室

## 简介
本程序用于进行平面连杆机构仿真。

1. 按照机架、杆、滑块等方式绘制机构。
2. 随时拖动任意构件得到机构状态。
3. 设定原动件及运动规律，测量机构位移、速度与加速度。
4. 保存/打开机构。
5. 导出GIF格式的机构仿真动画。
